# Vue Users CRUD

CRUD de usuarios hecho con Vue 3, Bootstrap, VeeValidate y Zod.

## Instalación

Clona el repositorio:

```bash
git clone <REPO_URL>
cd vue-users-crud
```

Instala dependencias:

```bash
npm install
```

## Ejecutar en desarrollo

```bash
npm run dev
```

Luego abrir en tu navegador:

```
http://localhost:5173
```

## Stack usado

- Vue 3 (Composition API)
- Bootstrap
- JavaScript
- Validación de formularios con VeeValidate y Zod

## Funcionalidades

- Crear usuarios
- Editar usuarios
- Eliminar usuarios
- Validación de formularios
- Manejo de estado reactivo
